# Visão Geral

Esta aplicação obtém imagens de gatos a partir da The Cat API e apresenta-as numa galeria com informação detalhada sobre cada raça.

## Utilizadores alvo
Qualquer pessoa que queira ver imagens aleatórias de gatos e aprender sobre as suas raças.

## Como funciona
A aplicação chama a The Cat API, obtém imagens com informação de raça associada, e apresenta-as numa RecyclerView. O utilizador pode ver detalhes de cada imagem, marcar favoritos e navegar entre a galeria e os favoritos através de uma barra de navegação inferior.
