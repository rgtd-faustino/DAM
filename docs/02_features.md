# Funcionalidades

1. Obter imagens de gatos a partir da The Cat API
2. Apresentar imagens numa grelha com scroll (RecyclerView)
3. Atualizar imagens com um botão
4. Mostrar indicador de carregamento enquanto as imagens são obtidas
5. Mostrar detalhes da imagem ao carregar nela
6. Marcar imagens como favoritas (fila FIFO, máximo 5)
7. Guardar em cache até 50 imagens para acesso sem ligação
8. Tratar erros da API de forma adequada
