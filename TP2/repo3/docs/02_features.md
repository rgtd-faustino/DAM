# Funcionalidades

1. Obter imagens de gatos a partir da The Cat API (com informação de raça)
2. Apresentar imagens numa grelha com scroll (RecyclerView)
3. Atualizar imagens com um botão
4. Mostrar indicador de carregamento enquanto as imagens são obtidas
5. Mostrar detalhes da imagem ao carregar nela (ID, URL, dimensões, raça, origem, temperamento, descrição, esperança de vida)
6. Marcar imagens como favoritas (fila FIFO, máximo 5)
7. Ecrã de favoritos com opção de remover
8. Botão de favorito no ecrã de detalhes com estado visual (verde/vermelho)
9. Guardar em cache até 50 imagens para acesso sem ligação
10. Tratar erros da API de forma adequada
11. Navegação por Bottom Navigation Bar entre Galeria e Favoritos
