# MyCatGalleryApp

Aplicação Android que apresenta uma galeria de imagens de gatos obtidas através da The Cat API.

## API utilizada
https://thecatapi.com

## Como executar
1. Abrir o projeto no Android Studio
2. Compilar o projeto com o Gradle
3. Executar num emulador ou dispositivo físico
